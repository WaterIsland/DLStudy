	＊活性化関数は、出力層も中間層も sigmoid
	＊訓練データ：input_data = [[0., 0.], [0., 1.], [ 1., 0.], [ 1., 1.]]

		＋出力層の更新式で対数を用いているため、'0'が出力される可能性を極力なくす必要があり、
